# Snake 

Popular snake game without any tweaks.

## Rules:

* Movement
    - "⬆️" - Up
    - "⬅️" - Left
    - "⬇️" - Down
    - "➡️" - Right
## Level:
    Intermediate

## Built with
* Python
    - Modules:
        - [Turtle](https://docs.python.org/3/library/turtle.html#module-turtle)
        - [Random](https://docs.python.org/3/library/random.html)
